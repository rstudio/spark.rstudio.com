# gf_unpersist


Unpersist the GraphFrame




## Description

Unpersist the GraphFrame





## Usage
```r
gf_unpersist(x, blocking = FALSE)
```




## Arguments


Argument      |Description
------------- |----------------
x | An object coercable to a GraphFrame (typically, a
``gf_graphframe``).
blocking | whether to block until all blocks are deleted
