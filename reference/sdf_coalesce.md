# `sdf_coalesce`

Coalesces a Spark DataFrame


## Description

Coalesces a Spark DataFrame


## Usage

```r
sdf_coalesce(x, partitions)
```


## Arguments

Argument      |Description
------------- |----------------
`x`     |     A `spark_connection` , `ml_pipeline` , or a `tbl_spark` .
`partitions`     |     number of partitions
